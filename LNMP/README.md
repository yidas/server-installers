INSTALLATION
------------

### Ubuntu 16.04 LTS

```
wget https://raw.githubusercontent.com/yidas/server-installers/master/LNMP/ubuntu16.04.sh -O installer
```

```
$ bash ./installer
```

> You can also make installer executable by `$ chmod +x ./installer` then `$ ./installer`.
