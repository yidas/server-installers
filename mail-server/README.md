### Postfix SMTP with Dovecot SASL in Ubuntu 16.04 LTS

> Postfix: 3.1.0  
> Dovecot: 2.2.22  

```
$ wget https://raw.githubusercontent.com/yidas/server-installers/master/mail-server/postfix-smtp-dovecot-sasl_ubuntu16.04.sh -O installer
$ bash installer
```
